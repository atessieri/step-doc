test.test_actor_dataset module

Unit tests for the ActorDataset module.

This module provides comprehensive unit tests for the ActorDataset and ActorDatasetConfig classes from the step_adapter package. The tests verify the proper functionality of actor-based dataset management, including CAM (Cooperative Awareness Messages) data handling, position information conversion, and time-based point management.

Test Coverage:

  • ActorDatasetConfig: Tests configuration validation, parameter initialization, and error handling for invalid configurations (negative values, zero frequency, etc.)

  • ActorDataset: Tests dataset operations including:

    • CAM data to position information conversion

    • Point dataset insertion and retrieval

    • Time-based cleanup of old points

    • Memory optimization verification (__slots__)

    • Error handling for edge cases (empty datasets, None values)

Key Features Tested:

  • Configuration validation with various parameter combinations

  • CAM message processing and position data extraction

  • Dataset point management with configurable time windows

  • Automated cleanup of expired data points

  • Integration with STEP cloud schemas for position and traffic data

  • Proper logging integration for debugging and monitoring

The test suite uses mocked dependencies to ensure isolated testing and predictable results through controlled time fixtures and mock logger instances. This allows for comprehensive verification of the ActorDataset behavior without external dependencies.
