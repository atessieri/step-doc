test.test_point_interpolation module

Test suite for point interpolation functionality.

This module provides comprehensive testing for the PointInterpolation and PointInterpolationConfig classes, which handle temporal and spatial interpolation of geographic position data points for tracking and navigation systems.

The module contains two main test classes:

  • TestPointInterpolationConfig: Tests configuration validation, parameter handling, and derived attribute computation for interpolation settings.

  • TestPointInterpolation: Tests the core interpolation logic, point insertion, timing validation, history management, and interpolated value generation.

Key Features Tested:

  • Configuration parameter validation and error handling

  • Point insertion with various timing scenarios (too short/long periods)

  • Geographic coordinate interpolation with proper handling of speed and heading

  • History queue management and point buffering

  • Interpolated value generation for continuous data streams

  • Edge case handling for boundary conditions and invalid inputs

The test suite uses pytest fixtures for mock objects and configured instances, parametrized tests for comprehensive scenario coverage, and follows the AAA (Arrange-Act-Assert) testing pattern throughout.

Note

This module requires the step_cloud_schemas package for geographic data types and the pytest ecosystem for test execution and mocking capabilities.
