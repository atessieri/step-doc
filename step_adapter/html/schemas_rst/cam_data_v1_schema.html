CAM Data V1 Schema

The CAM (Cooperative Awareness Message) Data Schema defines the structure for status information used in C-ITS system. This schema captures essential data including:

  • Station information and identification

  • Position and altitude data with confidence levels

  • Vehicle kinematics (speed, heading, acceleration)

  • Vehicle physical characteristics (length, width)

  • Dynamic parameters (curvature, yaw rate)

  • Vehicle role and drive direction

This schema defines a JSON Schema version 1. For further information, please refer to the python classes.

Example of the message:

{
  "creationTime": "2022-04-05T17:31:00.000Z",
  "stationInfo": {
    "stationId": 1236,
    "stationType": "passengerCar"
  },
  "position": {
    "latitude": 45.587878,
    "longitude": 60.458789,
    "positionConfidence": {
      "semiMajorAxisLengthConfidence": 12,
      "semiMinorAxisLengthConfidence": 123,
      "semiMajorAxisOrientation": 0
    },
    "altitude": {
      "value": 12000,
      "confidence": 50
    }
  },
  "heading": {
    "value": 123,
    "confidence": 50
  },
  "speed": {
    "value": 12367,
    "confidence": 50
  },
  "driveDirection": "forward",
  "vehicleLength": {
    "vehicleLengthValue": 123,
    "vehicleLengthConfidenceIndication": "noTrailerPresent"
  },
  "vehicleWidth": 16,
  "longitudinalAcceleration": {
    "value": 13,
    "confidence": 50
  },
  "curvature": {
    "value": 20,
    "confidence": 50
  },
  "curvatureCalculationMode": "yawRateNotUsed",
  "yawRate": {
    "value": 0,
    "confidence": 100
  },
  "vehicleRole": "default"
}
