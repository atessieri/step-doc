Test that the step_redis package exports all the expected symbols.

This test verifies:

  1. All expected exports are included in __all__

  2. All symbols listed in __all__ are actual attributes of the step_redis module

  3. No extra symbols are included in __all__ (length check)

Expected exports:

  • StepRedisPool: Main connection pool class

  • StepRedisPoolParams: Configuration parameters for the pool

  • StepRedisStream: Redis stream interface

  • __version__: Package version string
