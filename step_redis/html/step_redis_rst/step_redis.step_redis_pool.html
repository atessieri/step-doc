step_redis.step_redis_pool module

Redis connection pool management for asyncio applications.

This module provides a specialized Redis connection pool implementation that extends the BlockingConnectionPool from redis-py with enhanced features for asyncio applications. It offers both SSL and non-SSL connections to Redis servers with configurable TCP keepalive settings to maintain persistent connections, particularly useful for long-running services.

The module consists of two main components:

#. StepRedisPoolParams: A dataclass that encapsulates all configuration parameters for the Redis connection pool, including network settings, authentication credentials, SSL/TLS configuration, and pool behavior settings.

#. StepRedisPool: A connection pool implementation that leverages the asyncio capabilities of redis-py and provides context management for proper resource cleanup through the async context manager protocol.

Key Features:

  • Comprehensive SSL/TLS support for secure Redis connections

  • TCP keepalive configuration for connection stability

  • Configurable connection limits and timeouts

  • Proper resource management through async context managers

  • Automatic response decoding

  • Integrated logging support

Example

>>> import asyncio
... from logging import getLogger
... from step_redis.step_redis_pool import StepRedisPoolParams, StepRedisPool
...
... async def main():
...     # Configure Redis connection parameters
...     params = StepRedisPoolParams(
...         host="redis.example.com",
...         port=6379,
...         username="user",  # Optional
...         password="password",
...         db=0,
...         ssl_enable=True,
...         # ... other configuration parameters
...     )
...
...     logger = getLogger("redis_connection")
...
...     # Use the connection pool with async context manager
...     async with StepRedisPool(params=params, logger=logger) as pool:
...         # Create a Redis client using the pool
...         redis = Redis.from_pool(pool)
...         # Perform Redis operations
...         await redis.set("key", "value")
...         value = await redis.get("key")
...
... asyncio.run(main())

Bases: BlockingConnectionPool

A custom Redis connection pool class that extends BlockingConnectionPool and provides context management capabilities with async support. This class provides a blocking connection pool for Redis connections with support for both SSL and non-SSL connections. It configures connection parameters based on the provided StepRedisPoolParams and includes TCP keepalive settings for maintaining persistent connections. The class implements the asynchronous context manager protocol (__aenter__ and __aexit__), allowing it to be used with the async with statement for proper resource management.

Example

>>> redis_params = StepRedisPoolParams(
...    host="redis.example.com",
...    port=6379,
...    ssl_enable=True,
...    # ... other parameters
... logger = Logger("redis_connection")
... async with StepRedisPool(params=redis_params, logger=logger) as redis_pool:
...    # Use the redis pool here
...    # Connections are automatically managed and cleaned up

Notes

  • When SSL is enabled, the pool uses SSLConnection and configures all required SSL parameters

  • When SSL is disabled, the pool uses standard Connection

  • TCP keepalive settings are configured for both connection types

  • All responses are automatically decoded (decode_responses=True)
