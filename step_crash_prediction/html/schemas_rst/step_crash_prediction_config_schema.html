STEP Crash Prediction Config V1 Schema

The STEP Crash Prediction Config Version 1 Schema defines a comprehensive configuration structure for the STEP Crash Prediction. This JSON Schema is automatically generated from Pydantic models and provides strong typing, validation rules, and documentation.

Overview

The STEP Crash Prediction Configuration is organized hierarchically with the main StepCrashPredictionConfig class containing several nested configuration components for different aspects of the system:

  • Base Configuration

    • templateVersion: The version of the template (currently “1”)

    • receiveClientNumber: The number of clients to receive updates

    • reconnectInterval: Time between reconnection attempts (seconds)

  • Redis Client Configuration

    • Connection parameters for message broker integration

    • Security settings

  • Logging Configuration

    • Log handlers

    • Log levels

    • Formatting options

Features and Benefits

  • Type Validation: Strong typing and constraints via Pydantic fields

  • Immutable Design: Configuration is frozen after loading to prevent runtime errors

  • Comprehensive Documentation: Detailed field descriptions and examples

  • Default Values: Sensible defaults provided for all configuration options

  • Strict Validation: Prevents configuration errors before they cause runtime issues

Example

This example is created using JSON Schema version 1. For further information, please refer to the python classes.

Example of the configuration schema:

{
  "templateVersion": "1",
  "receiveClientNumber": 3,
  "reconnectInterval": 2,
  "redis": {
    "connection": {
      "host": "localhost",
      "port": 6379,
      "username": "username",
      "password": "password",
      "db": 0,
      "socketTimeout": 30,
      "socketConnectTimeout": 30,
      "socketKeepIdle": 30,
      "socketKeepAlive": 30,
      "socketKeepCnt": 5,
      "sslEnable": true,
      "sslClientKeyFile": "/path/to/client.key",
      "sslClientKeyFilePassword": "password",
      "sslClientCertFile": "/path/to/client.crt",
      "sslCertRequested": true,
      "sslCaCertsFile": "/path/to/ca_certs_file.crt",
      "healthCheckInterval": 20,
      "connectionAvailableTimeout": 10,
      "protocol": 3
    },
    "streams": {
      "outputs": {
        "producer": {
          "streamId": "stream id",
          "maxLength": 1000
        }
      },
      "input": {
        "streamId": "stream id",
        "groupId": "group id",
        "waitTimeout": 30000
      }
    }
  },
  "log": {
    "handler": {
      "handlerType": "file",
      "path": ".\\logFolder",
      "filenameTrailer": ".log",
      "append": true
    },
    "level": "INFO",
    "format": "%(asctime)s.%(msecs)03d [%(levelname)s][pid=%(process)d][module=%(module)s][function=%(funcName)s]: %(message)s",
    "dateFormat": "%Y-%m-%d %H:%M:%S"
  }
}
