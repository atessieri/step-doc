test.test_step_crash_prediction_config module

Test module for step_crash_prediction configuration components.

This module contains comprehensive unit tests for the step_crash_prediction configuration system, specifically testing the various logging handler classes, interval types, log modes, and the main StepCrashPredictionConfig class. The tests ensure proper functionality, default value settings, and schema/example file generation capabilities.

Key Testing Features:

  • Default configuration validation for all logging handler types

  • Parameterized testing for enumeration value conversions

  • File comparison testing for schema and template generation

  • Error handling testing for invalid input values

  • Temporary directory usage to avoid filesystem pollution

The tests utilize temporary directories for file operations and perform byte-level file comparisons to ensure generated schemas and examples match reference files.
