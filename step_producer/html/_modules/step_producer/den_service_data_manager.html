  1"""Decentralized Environmental Notification Message (DENM) Data Management Module.
  2
  3This module provides comprehensive management of DENM records for Intelligent Transportation
  4Systems (ITS). It implements a complete lifecycle management system for collision risk
  5notifications, including creation, storage, indexing, updating, and expiration of DENM
  6messages.
  7
  8The module consists of two main classes:
  9
 10    #. **DenmRecord**: A data container that encapsulates individual DENM messages along with
 11       their metadata such as reference time, unique record keys, and action ID indices. It
 12       provides comparison operators for sorting and organizing records chronologically, as well
 13       as hash functionality for efficient storage in dictionaries and sets.
 14    #. **DenServiceDataManager**: The primary service class that manages collections of DENM
 15       records. It handles the processing of station collision arrays, maintains multiple indices
 16       for efficient record retrieval, manages sequence number generation for action IDs, and
 17       provides automatic cleanup of expired records.
 18
 19Key Features:
 20
 21    - **Collision Detection Processing**: Converts station collision arrays into structured DENM
 22      records with proper ordering and unique identification
 23    - **Multi-Index Storage**: Maintains sorted lists and hash-based indices for optimal
 24      performance in both chronological and key-based lookups
 25    - **Automatic Expiration**: Implements time-based cleanup of outdated records based on
 26      configurable delay thresholds
 27    - **Sequence Management**: Handles automatic generation and wrap-around of sequence numbers
 28      for action IDs
 29    - **Comprehensive Logging**: Provides detailed debugging and operational logging throughout
 30      the record lifecycle
 31
 32The module is designed to handle high-frequency collision detection scenarios while
 33maintaining efficient memory usage and providing fast access patterns for real-time
 34ITS applications.
 35
 36Example Usage:
 37    >>> manager = DenServiceDataManager(
 38    ...     station_id=12345,
 39    ...     time_after_collision=5000,
 40    ...     denm_last_update_max_delay_ms=10000,
 41    ...     denm_check_last_update_max_interval=30000,
 42    ...     logger=logging.getLogger(__name__)
 43    ... )
 44    ... records = manager.insert_station_collision_array(collision_array)
 45    ... expired_records, oldest = manager.get_denm_records_clean_reference_expired()

315class DenServiceDataManager:
316    """Manages Decentralized Environmental Notification Message (DENM) data for
317    collision risk events. This class is responsible for creating, storing, updating and
318    managing DENM records related to potential vehicle collisions. It maintains various
319    indexes for efficient retrieval and processing of DENM records.
320    DenServiceDataManager organizes collision data into structured records, handles the
321    lifecycle of these records including creation, update and expiration, and provides
322    methods to access and manipulate the stored data.
323
324    Attributes:
325        station_id: The ID of the station generating these DENM messages.
326        _sequence_number: Counter for generating unique sequence numbers for action IDs.
327        time_after_collision: Time to continue broadcasting a DENM after collision (in ms).
328        denm_last_update_max_delay_ms: Maximum allowed delay for DENM record updates.
329        denm_check_last_update_max_interval: Maximum interval for checking record updates.

461    def insert_station_collision_array(
462        self, station_collision_array: StationCollisionArrayV1
463    ) -> list[DenmRecord]:
464        """Processes a StationCollisionArrayV1 and creates or updates DENM records for
465        station pairs.
466
467        This method takes a StationCollisionArrayV1 object, orders the stations it contains,
468        and creates DENM (Decentralized Environmental Notification Message) records for each
469        pair of stations where the first station is used as a position reference.
470
471        Args:
472            station_collision_array (StationCollisionArrayV1): The collision array containing
473                stations information and detection time.
474
475        Returns:
476            list[DenmRecord]: A list of new or updated DENM records created during processing.
477
478        Notes:
479            - For each pair of stations, a unique record key is created based on station IDs and
480              object IDs.
481            - If a record with the same key already exists, its action ID is reused, otherwise a
482              new one is created.
483            - The sequence number is incremented for each new action ID.
484            - The method maintains several indexes for efficient record retrieval and management.
485            - The reference of the DENM is always the first station in the ordered list.
486            - This method is thread-safe and uses internal locking to ensure data consistency
487              during the cleanup operation.
488        """
489        detection_time_its = ItsTime.create_from_iso_time(
490            time_str=station_collision_array.detectionTime
491        )
492        ordered_stations = self._station_order(
493            stations=station_collision_array.stations
494        )
495        position_reference: Optional[PositionInfo] = None
496        denm_list: list[DenmRecord] = []
497        with self._lock:
498            for station in ordered_stations:
499                self._logger.debug(
500                    msg=f"Processing station: {station.stationInfo.stationId}, objectId: {station.objectId}"  # pylint: disable=line-too-long
501                )
502                if position_reference is None:
503                    position_reference = station
504                else:
505                    # Create a unique record key based on the involved stations
506                    record_key = (
507                        position_reference.stationInfo.stationId,
508                        (
509                            position_reference.objectId
510                            if position_reference.objectId
511                            else 0
512                        ),
513                        station.stationInfo.stationId,
514                        station.objectId if station.objectId else 0,
515                    )
516                    self._logger.debug(msg=f"record key: {record_key}")
517                    # Create or retrieve the action_id
518                    action_id: ActionId
519                    if record_key in self._record_key_index:
520                        action_id = self._record_key_index[
521                            record_key
522                        ].denm.managementContainer.actionId
523                        self._logger.debug(
524                            msg=f"Found existing record {record_key} action id: {action_id}"
525                        )
526                        self._denm_sorted_records.discard(
527                            value=self._record_key_index[record_key]
528                        )
529                    else:
530                        action_id = ActionId(
531                            originatingStationId=self._station_id,
532                            sequenceNumber=self._sequence_number,
533                        )
534                        self._logger.debug(
535                            msg=f"New record {record_key} action id: {action_id}"
536                        )
537                        # Wrap around at 65536
538                        self._sequence_number = (self._sequence_number + 1) % 65536
539                    # Create the new DENM record
540                    record = self._create_denm(
541                        record_key=record_key,
542                        action_id=action_id,
543                        detection_time_its=detection_time_its,
544                        station_collision_array=station_collision_array,
545                        position_reference=position_reference,
546                        station=station,
547                    )
548                    # Create or update the DENM record
549                    self._record_key_index[record_key] = record
550                    self._action_id_index[
551                        (action_id.originatingStationId, action_id.sequenceNumber)
552                    ] = record
553                    self._denm_sorted_records.add(value=record)
554                    denm_list.append(record)
555        return denm_list

557    def get_denm_records_clean_reference_expired(
558        self,
559    ) -> tuple[list[DenmRecord], Optional[DenmRecord]]:
560        """Remove expired DENM records and those with outdated reference times.
561
562        This method performs cleanup of DENM (Decentralized Environmental Notification Message)
563        records by removing those that have either expired based on their validity duration
564        or have reference times older than the maximum allowed delay.
565
566        The method processes records in chronological order (oldest first) and:
567
568            #. Removes records that have exceeded their validity duration
569            #. Removes records with reference times older than the minimum threshold
570            #. Stops processing when it encounters a valid, non-expired record
571            #. Updates internal indexes and logs removal operations
572
573        Returns:
574            tuple[list[DenmRecord], Optional[DenmRecord]]: A tuple containing:
575
576                - A list of removed DENM records that were expired or had outdated reference times
577                - The oldest remaining valid record, or None if no valid records remain
578
579        Notes:
580            This method is thread-safe and uses internal locking to ensure data consistency
581            during the cleanup operation.
582        """
583        now = ItsTime()
584        reference_time_min = now - self._denm_last_update_max_delay_ms
585        records_to_remove: list[DenmRecord] = []
586        with self._lock:
587            while True:
588                oldest_record: Optional[DenmRecord]
589                try:
590                    oldest_record = cast(
591                        DenmRecord, self._denm_sorted_records.pop(index=0)
592                    )
593                except IndexError:
594                    oldest_record = None
595                    break
596                validity_duration = (
597                    oldest_record.denm.managementContainer.validityDuration
598                    if oldest_record.denm.managementContainer.validityDuration
599                    else 1000
600                )
601                if now <= oldest_record.reference_time + validity_duration:
602                    if oldest_record.reference_time >= reference_time_min:
603                        self._denm_sorted_records.add(value=oldest_record)
604                        break
605                    records_to_remove.append(oldest_record)
606                del self._record_key_index[oldest_record.record_key]
607                del self._action_id_index[oldest_record.action_id_index]
608                self._logger.debug(
609                    msg=f"Removing DENM: key {oldest_record.record_key}, "
610                    f"action ID {oldest_record.action_id_index}, "
611                    f"reference time {oldest_record.reference_time} ms"
612                )
613        return (records_to_remove, oldest_record)

615    def get_interval_oldest_update_ms(
616        self, oldest_record: Optional[DenmRecord] = None
617    ) -> int:
618        """Calculates the time interval in milliseconds since the oldest DENM record was
619        updated.
620
621        If an `oldest_record` is provided, the interval is calculated from its reference time.
622        Otherwise, the interval is calculated from the reference time of the first record in
623        the `_denm_sorted_records` list. If no records are available, returns the maximum
624        allowed interval (`_denm_check_last_update_max_interval`).
625
626        Args:
627            oldest_record (Optional[DenmRecord]): The DENM record to use as the oldest record.
628                If None, the method uses the first record in `_denm_sorted_records`.
629
630        Returns:
631            int: The interval in milliseconds since the oldest DENM record was updated.
632
633        Notes:
634            This method is thread-safe and uses internal locking to ensure data consistency
635            during the cleanup operation.
636        """
637        with self._lock:
638            try:
639                if oldest_record is not None:
640                    record = oldest_record
641                else:
642                    record = cast(DenmRecord, self._denm_sorted_records[0])
643                interval = ItsTime().its_ms - record.reference_time.its_ms
644            except IndexError:
645                self._logger.debug(msg="No DENM records found")
646                interval = self._denm_check_last_update_max_interval
647            if interval > self._denm_check_last_update_max_interval:
648                self._logger.debug(
649                    msg=f"Oldest DENM record update interval {interval} ms "
650                    f"exceeded maximum {self._denm_check_last_update_max_interval} ms"
651                )
652                interval = self._denm_check_last_update_max_interval
653        self._logger.debug(msg=f"Oldest DENM record update interval: {interval} ms")
654        return interval
