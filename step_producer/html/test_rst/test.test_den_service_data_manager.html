test.test_den_service_data_manager module

Unit tests for the DEN (Decentralized Environmental Notification) Service Data Manager.

This module contains comprehensive test suites for the DenServiceDataManager and DenmRecord classes, which are responsible for managing Decentralized Environmental Notification Messages (DENM) in the STEP producer system. DENM messages are part of the Cooperative Intelligent Transport Systems (C-ITS) standard and are used to communicate hazardous events and situations on the road.

Test Classes:
TestDenmRecord: Tests for the DenmRecord class that represents individual DENM records

with functionality for:

  • Record initialization with DENM data and keys

  • Equality and comparison operations for sorting and deduplication

  • Hash computation for use in collections

  • String representation for debugging

  • Property access for record attributes

TestDenServiceDataManager: Tests for the DenServiceDataManager class that manages

collections of DENM records with functionality for:

  • Initialization of data structures and configuration

  • Station ordering algorithms for consistent record generation

  • DENM record creation from collision detection data

  • Station collision array insertion and processing

  • Expired record cleanup and management

  • Update interval calculations for periodic operations

  • Record removal operations

Key Features Tested:

  • DENM record lifecycle management (creation, storage, expiration, removal)

  • Station collision array processing and conversion to DENM format

  • Time-based operations including expiration checking and update intervals

  • Index maintenance for efficient record lookup by action ID and record key

  • Sequence number management for unique record identification

  • Configuration handling for timing parameters and station identification

The tests use pytest fixtures to create mock objects and test data, ensuring isolated and reproducible test execution. Parameterized tests verify behavior across different time scenarios and configuration parameters.
