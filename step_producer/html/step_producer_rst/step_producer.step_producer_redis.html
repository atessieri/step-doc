step_producer.step_producer_redis module

STEP Producer Redis Integration Module.

This module provides specialized Redis integration for STEP (Secure Transport Events Platform) Producer services. It defines Redis connection pools and streams optimized for handling ITS (Intelligent Transportation Systems) messages, with a focus on DENM (Decentralized Environmental Notification Message) data.

Key Components:

  1. StepProducerRedisPool: A specialized Redis connection pool for STEP Producers that limits the maximum number of connections. Designed to be used as an async context manager for efficient resource management.

  2. StepProducerRedisStream: A Redis stream implementation tailored for STEP Producer service that handles:

    • Creating and managing client input streams

    • Publishing ITS messages (DENM data) to client streams

    • Receiving and validating ITS messages from Redis streams

The module integrates with the STEP ecosystem by leveraging the step_cloud_schemas for message structures and step_redis for base Redis functionality, providing a seamless interface for STEP Producers to interact with Redis for real-time ITS message processing.

Example Usage:
>>> # Set up Redis connection and stream
... async with StepProducerRedisPool(config, logger) as redis_pool:
...     stream = StepProducerRedisStream(consumer_name, stream_config, logger, redis_pool)
...     async with stream:
...         # Create input stream
...         await stream.create_client_input_stream()
...
...         # Receive and process messages
...         message = await stream.receive_its_message()
...
...         # Publish processed messages
...         await stream.publish_its_message(denm_data)
