step_producer.step_producer_config module

STEP Producer configuration module.

This module provides configuration classes and utilities for the STEP Producer component. It leverages Pydantic for configuration validation, serialization, and schema generation.

The configuration covers several key aspects:

  • Redis connection settings and stream configurations

  • Logging configuration with various handlers (stream, file, null)

  • Producer instance settings including station ID and timing parameters

The module is structured with a hierarchical configuration model:

  • StepProducerConfig: Top-level configuration class

  • RedisClientConfig: Configuration for Redis client connections

  • RedisStreams: Configuration for Redis streams (input/output)

  • LogConfig: Configuration for logging subsystems

Each configuration class provides:

  • Strong typing and validation via Pydantic

  • Default configuration instances via get_default() methods

  • JSON schema generation capabilities

  • Immutable (frozen) instances to prevent accidental modifications

Example

>>> # Create a default configuration
... config = StepProducerConfig.get_default()
...
... # Print the JSON schema to a file
... config.print_config_schema(Path("./config"))
