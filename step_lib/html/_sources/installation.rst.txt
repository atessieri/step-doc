============
Installation
============

Step Lib works on Windows, Linux and MacOS. Please follow the instructions below
to build and run an example in your specific environment.

--------------------
Installing from PyPI
--------------------

The fastest way to get going with SimplePyBLE is to install it from PyPI: 

.. code-block:: bash

   $pip install step_lib

Please take into consideration that today the library is in an alpha version on Test PyPI.
So the command to install the library is:

.. code-block:: bash

   $pip install --extra-index-url https://test.pypi.org/simple/ step_lib
