step_lib.geonet.basic_header module

GeoNetworking Basic Header Implementation Module.

This module provides classes and utilities for working with the GeoNetworking Basic Header as specified in ETSI TS 103 836-4-1 V2.1.1. The Basic Header is the first part of every GeoNetworking packet and contains essential routing and protocol information.

The module includes:

  • Enumerations for protocol versions and header types (GeonetVersion, NextHeader)

  • Time management utilities for packet lifetime (LifeTimeBase, LifeTime)

  • Implementation of the BasicHeader class with serialization/deserialization functionality

The Basic Header structure contains:

  • Protocol version (4 bits)

  • Next Header type indicator (4 bits)

  • Reserved field (8 bits)

  • Lifetime value (8 bits)

  • Remaining hop limit (8 bits)

This module supports parsing from byte streams and conversion to byte representation for network transmission in GeoNetworking applications.

Parse a BasicHeader from a byte stream.

This method reads from the provided byte stream to construct a BasicHeader object. The byte stream is expected to contain:

  • 1 byte in msb:

    • 4 bits for version big endian

    • 4 bits for next_header big endian

  • 1 byte reserved (skipped)

  • LifeTime object bytes (read via LifeTime.from_byte_stream)

  • 1 byte for remaining_hop_limit

to_bytes() → bytes[source]

Convert the basic header to bytes.

This method serializes the basic header into a byte stream according to the GeoNet protocol. The serialization includes most significant bit (MSB) ordering for the first byte, which contains:

  • Version (4 bits)

  • Next Header type (NH) (4 bits)

  • Reserved field (8 bits, set to 0x00)

  • Lifetime (LT) (8 bits)

  • Remaining hop limit (RHL) (8 bits)

class step_lib.geonet.basic_header.LifeTime(*, value: int = 950)[source]

Bases: object

A class for managing and representing lifetime values in GEONET protocol. The LifeTime class is responsible for handling lifetime values, which are durations expressed in a compact form using a combination of a multiplier and a base value. This encoding allows for representing a wide range of time intervals using a single byte. This class provides functionality to:

  • Create lifetime values from byte streams

  • Convert lifetime values to their byte representation

  • Calculate appropriate base values and multipliers for efficient encoding

value

The actual lifetime value, with a default of 950.

Type:

int

static from_byte_stream(stream: BytesIO) → LifeTime[source]

Create a LifeTime object from a byte stream.

This method reads one byte from the provided byte stream and interprets it as follows starting from the most significant bit (MSB) big endian:

  • The first 6 bits represent the multiple value

  • The last 2 bits represent the LifeTimeBase

The final LifeTime value is calculated by multiplying the multiple value with the corresponding base value derived from the LifeTimeBase.

Parameters:

stream (BytesIO) The byte stream to read from

Returns:

A new LifeTime object with the calculated value

Return type:

LifeTime

Note

This method advances the streams position by one byte.

to_bytes() → bytes[source]

Converts the lifetime value to a byte representation.

The method encodes the lifetime value using a combination of a multiplier and a base value. It first determines the appropriate base by iterating through available lifetime base values until finding one that, when multiplied by the maximum allowed multiplier, is greater than or equal to the current lifetime value.

Then it calculates the actual multiplier for the chosen base (capped to MAX_MULTIPLIER) and creates a bit-stream with 6 bits (MSB big endian) for the multiplier and 2 bits (big endian) for the base identifier.

Returns:

A bytes object containing the encoded lifetime value (1 byte total).

Return type:

bytes
