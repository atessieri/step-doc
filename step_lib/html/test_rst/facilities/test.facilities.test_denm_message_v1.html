test.facilities.test_denm_message_v1 module

Test module for the DenmMessageV1 class.

This module contains the test suite for validating the functionality of the DenmMessageV1 class, which handles Decentralized Environmental Notification Messages (DENM) in vehicular communications.

The test suite covers:

  • Message initialization and validation

  • Data format conversions (ASN.1, ORM, Dictionary, JSON)

  • Message property access and manipulation

  • Station ID modifications

  • Error handling for invalid operations

The tests use fixtures to provide sample DENM messages in various formats:

  • ASN.1 structured dictionary

  • ORM (DenmDataV1) objects

  • Python dictionaries

  • JSON strings

Each test case validates specific aspects of the DenmMessageV1 class to ensure proper handling of DENM messages throughout the STEP library ecosystem.

class test.facilities.test_denm_message_v1.TestDenmMessageV1[source]

Bases: object

The class is a test suite for the DenmMessageV1 class, which handles Decentralized Environmental Notification Messages (DENM) in vehicular communications. The test suite ensures that DenmMessageV1 properly manages DENM message data including:

  • Initialization with different message formats

  • Conversion between different data representations (ASN.1, ORM, Dict, JSON)

  • Property access and validation

  • Station ID modification

  • Message presence detection

  • Error handling when operations are performed on empty messages

The test fixtures provide sample DENM messages in various formats:

  • message_asn: A dictionary representing an ASN.1 structured DENM message

  • message_orm: A DenmDataV1 object with complete DENM data

  • message_orm_empty: A DenmDataV1 object with minimal DENM data

  • message_dict: A Python dictionary representation of a DENM message

  • message_json: A JSON string representation of a DENM message

These fixtures enable comprehensive testing of data conversion, validation, and manipulation functionality within the DenmMessageV1 class.
